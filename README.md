# 멀티 채팅봇   

Study Mini Project


## 1. 개발 환경
운영체제 : windows OS   
개발도구 :  eclipse   
Language : Java   

## 2. 구현 기능
Client : 서버 연결, 메세지 읽기, 서버에 메세지 보내기, 채팅 종료
Server : 서버 시작, 메세지 읽기, 귓속말, 클라이언트에게 메세지 보내기, 알림, 서버 종료
